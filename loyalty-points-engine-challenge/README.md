# ☕ Loyalty Points Engine

A Python-based logic engine that calculates customer loyalty rewards based on spending habits.

## 🚀 Features
- **Point Calculation:** Converts whole-dollar amounts into points (3 pts per $1).
- **Tier System:** Automatically assigns customers to **Bronze**, **Silver**, or **Gold** tiers.
- **Summary Reporting:** Generates a formatted financial summary.

## 🛠️ Logic Breakdown
The engine uses a `for` loop to aggregate purchase history and applies conditional logic (`if/elif/else`) to determine the final membership status.

## 📝 How to Run
1. Ensure you have Python installed.
2. Run `python loyalty_engine.py`
