# 🧮 Math Multiplication Tutor

A lightweight, interactive command-line tool built in Python to help users practice their multiplication tables. This project was developed as part of my Python learning journey.

## 🚀 Features
* **Customizable Practice:** Choose how many questions you want and the maximum number range.
* **Performance Tracking:** Calculates total time taken and your average speed per question.
* **Score Summary:** Provides a final percentage and a side-by-side comparison of the correct answers versus your answers.
* **Clean Feedback:** Visual indicators (✅/❌) to make the final review easy to read.

## 🛠️ How to Run
1. Ensure you have **Python 3.x** installed.
2. Download the `math_tutor.py` file.
3. Open your terminal or command prompt.
4. Run the script:
   ```bash
   python math_tutor.py
